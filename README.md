## logseq-tidy-blocks-plugin

## Features
### Remove whitespace from _one_ block or _multiple_ blocks in 3 ways:
- Block context menu (right-click on bullet)
- Slash (/) command
- Keyboard shortcut
    - Usage: Click on the block (as if to edit it) â†’ Activate the keyboard shortcut (can be configured in the [settings](#settings))
